🔐 Java CWE Security Skills

Deterministic security remediation skills for AI coding agents mapped to MITRE CWE vulnerabilities

Java Security AI/ML DevSecOps Open Source
⭐ Star on GitHub 📖 View Docs

The Problem

Modern SAST tools like Checkmarx, SonarQube, Snyk, and CodeQL excel at detecting vulnerabilities but often lack deterministic remediation patterns.

Developers must manually research each CWE, understand the vulnerability, and implement fixes—consuming hours of engineering time per vulnerability.

The Solution

This repository provides AI-readable remediation skills that enable coding agents to:

🔍

Detect

Identify vulnerable code patterns using grep/regex commands

🗺️

Map

Link patterns to specific MITRE CWE weaknesses

🔧

Fix

Apply deterministic secure code transformations

✅

Verify

Confirm fixes through structured verification rules
